# MBTI Chat

> **MBTI AI counselor** — Chat with AI counselors typed by MBTI personality, powered by ChatGPT API.

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Jekyll](https://img.shields.io/badge/Jekyll-static-red?logo=jekyll)](https://jekyllrb.com/)

---

## Overview

**MBTI Chat** is a Jekyll-based static site that uses the ChatGPT API to simulate AI counseling conversations with characters typed by 16 MBTI personality types. Each AI counselor speaks and reasons in the style of a specific MBTI type.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Generator** | Jekyll |
| **Styling** | SCSS |
| **AI** | OpenAI ChatGPT API |
| **Hosting** | GitHub Pages |

## Local Development

```bash
git clone https://github.com/moony01/mbtichat.git
cd mbtichat
bundle install
bundle exec jekyll serve
```

Open http://localhost:4000/
