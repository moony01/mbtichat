mbtiChat 应用程序是一个允许用户根据 MBTI 的 16 种类型与人工智能进行自由对话的应用程序。该应用程序的开发是基于用户对不同 MBTI 类型的特征、不同类型之间的关系以及不同类型的思维和行为的好奇。现在让我们来了解一下如何使用 mbtiChat 应用程序。

## 应用程序功能和页面介绍
除了与人工智能对话的功能外，我们的应用程序还具有更多详细的功能和指南、博客帖子功能。mbtiChat 应用程序的功能和页面主要包括**与人工智能对话功能、博客**。除此之外，还有您当前正在查看的用户指南页面和语言选择功能等详细功能，但我们将在了解这两个主要功能和页面的同时详细了解其他功能。

### 1. 与人工智能对话功能
![MBTI 人工智能对话语言选择](/images/userGuide/userguide-1.jpg)

与人工智能对话功能允许您从主页开始与经过 MBTI 学习的角色之一进行对话，方法是点击 [开始对话] 按钮。人工智能是基于 **gpt-3.5** 版本的 API 开发的。首先，为了与人工智能对话，您需要**选择语言以更改应用程序的语言**。

![MBTI 人工智能对话角色选择和开始对话按钮](/images/userGuide/userguide-2.jpg)

在主页上，您可以点击角色底部的 [开始对话] 按钮与人工智能进行对话。这些角色中的每一个都已经学习了一个 MBTI 类型。然后，让我们通过点击 [开始对话] 按钮与我们选择的角色进行对话。我们已经进入到对话页面，您可以在下面的图片中看到角色介绍和请求您提供您的 MBTI 类型的消息。

![MBTI 与人工智能对话页面](/images/userGuide/userguide-3.jpg)

您可以向人工智能提问而不提供您的 MBTI 类型，但如果您想了解特定类型的特征、根据他们的行为推测您的类型或了解您的类型与 AI 类型之间的匹配程度等问题，则需要向 AI 提供您的 MBTI 类型。您可以问角色：“INTP 的特征是什么？” 或者 “我和我的朋友一起看了一部悲伤的电影，但我的朋友一点都不难过。我的朋友可能是 INTP 吗？” 或者 “请告诉我关于我和我的朋友（INTP）之间的匹配程度。” 通过这样的提问，您可以了解**特定类型的特征**、**根据行为推断您的类型**以及**了解您的类型和朋友的类型之间的匹配程度、优势和劣势**。另外，如上图所示，由于 mbtiChat 应用程序是通过广告和赞助运营的，因此在与角色对话的过程中会显示一条引导消息。

![与人工智能对话后的功能：分享、表情、评论](/images/userGuide/userguide-7.jpg)

与人工智能对话有帮助吗？那么请与朋友分享 mbtiChat 应用程序并留下许多评论！**请放心，该应用程序的对话内容不会被保存。**

### 2. 博客
![mbtiChat 应用程序的博客页面](/images/userGuide/userguide-8.jpg)

mbtiChat 应用程序不仅具有功能，还包括博客部分。博客文章主要涉及在开发 mbtiChat 应用程序过程中使用的开发技巧。

## 应用程序的技术分析和介绍
mbtiChat 应用程序是一个静态网页应用程序。静态网页应用程序的内容是固定的，具有受限的动态功能和交互。尽管 mbtiChat 应用程序在规模上不大，但它具有博客功能和语言选择功能等超出静态网页应用程序能力的功能。因此，我使用了 **Jekyll** 作为创建静态网页的框架。

Jekyll 根据指定的模板和布局生成静态的 HTML 文件。这样可以有效地重用重复的元素，并应用一致的设计和布局。

mbtiChat 应用程序的主要功能是与人工智能对话。人工智能使用 **gpt-3.5** 版本的 API。为了与 GPT API 进行通信，我使用 **Node.js** 作为服务器。移动应用程序是使用 **React-Native-Webview** 创建的响应式 Web 技术的移动应用程序。
